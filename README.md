# Go-Vue-Admin

## 介绍
Go-Vue-Admin 是一套权限管理系统，基于 golang、 gin、go-jwt、validator、go-casbin、mysql、redis、gorm、vue、axios、vue-router、vuerx、element ui 、 开发，包括 golang服务端、 Vue后台管理员前端，主要功能有用户管理、角色管理、菜单管理、接口管理、字典管理、操作日志、七牛云文件上传等。
